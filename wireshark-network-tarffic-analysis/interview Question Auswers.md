1. What is Wireshark used for?
Wireshark is a network protocol analyzer used to capture and inspect network traffic in real time.

2. What is a packet?
A packet is a small unit of data transmitted across a network.

3. How to filter packets in Wireshark?
Use display filters such as tcp, udp, dns, http, or tls in the filter bar.

4. What is the difference between TCP and UDP?
TCP is connection-oriented and reliable. UDP is connectionless and faster but does not guarantee delivery.

5. What is a DNS query packet?
A DNS query packet is a request sent to a DNS server to translate a domain name into an IP address.

6. How can packet capture help in troubleshooting?
It helps identify connectivity problems, packet loss, protocol errors, and abnormal traffic.

7. What is a protocol?
A protocol is a set of rules that devices follow to communicate over a network.

8. Can Wireshark decrypt encrypted traffic?

Only in certain cases when the required decryption keys or session secrets are available. Otherwise encrypted traffic remains unreadable.
